# Project Overview: NEXUS AI Shop (ai-autonomous-webshop)

**Purpose:**
A production-ready, autonomous AI-driven commerce platform designed for 2026. It aims to combine high-conversion retail design with a fully autonomous backend.

**Current Tech Stack (To be migrated):**
*   **Frontend:** React 19, Vite 7, TailwindCSS 4, Framer Motion, Lucide React.
*   **State Management:** Zustand.
*   **Backend/DB:** Supabase (PostgreSQL), Edge Functions.
*   **Automation:** n8n.
*   **AI:** Puter.js, Nvidia/Kimi integration.
*   **Language:** TypeScript.

**Target Tech Stack (Migration Goal):**
*   **Backend:** Go (Golang) - Complete migration from Supabase Edge Functions/Monolith.
*   **Frontend:** React (Redesigned for 2026 Best Practices).
*   **Infrastructure:** Docker, Modular Architecture.

**Key Features:**
*   Unified Global Shop.
*   AI Orchestration (Chatbots, Admin Automation).
*   Autonomous Operations (Fulfillment, Workflow Engine).
*   Real-time Analytics.
